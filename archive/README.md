# الأرشيف المرجعي

هذه الملفات أُعيدت كمواد داعمة للمشروع فقط، وتم وضعها داخل `archive/` لتجنب أي تعارض مع ملفات التشغيل الحالية.

- `AI_ANALYSIS_SETUP.md` — ملاحظات قديمة تخص إعدادات تكامل الذكاء الاصطناعي
